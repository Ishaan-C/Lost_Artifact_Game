# Text-Based Adventure Game: The Lost Artifact of Olympus

Welcome to *The Lost Artifact of Olympus*, a Python-based text adventure game that takes you on a thrilling journey through ancient Roman lands. Solve puzzles, battle fearsome monsters, and explore mysterious rooms to recover a powerful artifact and restore hope to a dying world.

---

## Overview

This game is a classic text-based adventure built with Python, featuring:  
- Multiple interconnected rooms with unique challenges  
- Puzzle-solving via riddles and number guessing  
- Turn-based combat against mythical monsters  
- An in-game shop to purchase weapons and healing items  
- Player stats management: health, coins, and inventory  

---

## Getting Started

### Prerequisites

- Python 3.x installed on your machine  
- Basic command-line familiarity
